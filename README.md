# Jellyfin Wheel Control v1.0.0

**Enable mouse wheel page navigation in Jellyfin Web slideshow.**

---

## 概要

Jellyfin Web クライアントで画像や漫画を閲覧する際、マウスホイールでページ送りを可能にする Chrome 拡張機能です。  

公式 Web UI ではホイール操作がページ送りに対応していないため、より直感的な閲覧体験を提供します。

---

## 特徴

- **Jellyfin Web 専用**  
  DOM 構造を確認して動作するため、他サイトでは誤作動しません。
- **マウスホイールで前/次ページ**  
  左右の矢印ボタン操作に対応。
- **ページ端で自動無効化**  
  最初/最後ページで無意味なクリックを防止。
- **Jellyfin公式UX追従**  
  ズーム中でも公式の仕様に従い、ページ送りは制御しません。
- **軽量・依存なし**  
  Manifest v3 / content.js だけで動作。

---

## インストール

1. Chrome / Chromium 系ブラウザを開く
2. `chrome://extensions/` にアクセス
3. デベロッパーモードを有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このリポジトリを選択

---

## 使い方

1. Jellyfin Web クライアントで画像または漫画を開く
2. マウスホイールを上下に動かすとページ送り
3. ページ端では自動的に操作が無効化されます

---

## 注意点

- Jellyfin Web 専用の拡張です。他のサイトでは動作保証しません。
- ズーム操作中も公式仕様に従うため、ホイールでページが進む場合があります（拡張は制御しません）。
- 将来的な Jellyfin UI の変更により動作が変わる可能性があります。
